### The CI flake (original ask)
`SimplePluginLoaderTest` failed in GitHub Actions: the `ne` case measured 2054µs against the
1000µs assertion. Root cause: 2 warmup iterations + a single-shot `nanoTime` measurement is
cold-start latency, not steady-state — one JIT/GC spike fails the test. Fix: 200 warmup
iterations, assert on the **median of 20 samples**. Threshold unchanged.

### Review findings fixed (all 43 plugins read)
- **`EqualsOperator`/`NotEqualsOperator`**: `reduce(Objects::equals)` is wrong for 3+ inputs —
  it compares the previous *Boolean result* against the next element. Replaced with
  `allMatch`/`anyMatch` against `input[0]`.
- **`DivideNumbers`**: `divideByZeroCheck` ran on all args, so `div(0, 5)` threw — now only
  divisors (`input[1..n]`) are checked, after the arity check.
- **`DateGenerator`**: `dateTime(pattern, zone)` silently dropped the pattern
  (`input.length == 1` → `>= 1`).
- **`SubstringPlugin`**: empty input threw raw `ArrayIndexOutOfBoundsException`.
- **`UniqueSet`**: `HashSet` → `LinkedHashSet` for deterministic order.
- **`InputValidation`** (behavior change, Eric-directed): when `required` is not configured the
  field is **optional — null passes validation** (returns null, or `true` under `evaluate`).
  Required+null under `evaluate` returns `false` instead of throwing. Two `FlowTests` cases
  (`inputValidationTest5`, `inputValidationTest14`) asserted the old always-throw behavior and
  were flipped to expect success. Also refactored `calculate()` under SonarQube's
  cognitive-complexity 15 (guard clause + extracted `resolveType`/`resolveNullValue`).
- **Exception-type unification** (Eric patched `SimplePluginUtils`, I aligned the 5 arithmetic
  plugins): all input validation throws `IllegalArgumentException`; the `IllegalStateException`s
  were IDE auto-complete slips.
- Housekeeping: copy-paste error messages, redundant `toLowerCase`, missing Apache headers on
  `StartsWith`/`EndsWith`/`Includes` operators.
- **Test coverage**: `SimplePluginLoaderTest` extended from 30 to 46 cases — all 43 plugins now
